Команды бота
На данный момент Микки имеет 37 команд!
❯ Развлечения

1. cat - Рандом картинка с котом.
2. dog - Рандом картинка с собакой.
3. meme - Рандом картинка с мемом.
❯ Модерация

1. addrole - Выдать роль пользователю.
2. ban - Забанить пользователя.
3. clear - Очистить сообщения.
5. kick - Кикнуть/Выгнать пользователя.
6. poll - Создать голосование.
7. removerole - Забрать роль у пользователя.
8. unban - Разбанить пользователя(Требуется ID Пользователя).
9. warn - Выдать предупреждение пользователю.
10. unwarn - Забрать все предупреждения у пользователя.
11. warnlevel - Посмотреть количество предупреждений.
❯ Кастомизация

1. setwchannel - Установить приветственный канал.
2. setbchannel - Установить канал для прощаний.
3. setilchannel - Установить канал для логов об инвайтах.
4. setprefix - Установить новый префикс сервера.
❯ Остальное

1. help - Список команд.
2. math - Решить математический пример.
3. userinfo - Информация пользователя.
4. ping - Пинг бота.
5. report - Подать жалобу на пользователя.
6. serverinfo - Информация сервера.
7. uptime - Время работы бота.
8. config - Конфиг сервера.
9. weather - Погода в городе/стране.
❯ Поддержка

1. bugreport - Уведомить создателя о баге.
2. call - Вызвать создателя на сервер.
❯ Музыка(Новинка)

1. play - Включить музыку.
2. stop - Остановить всю музыку.
3. skip - Пропустить песню.
4. pause - Поставить музыку на паузу.
5. resume - Возобновить музыку.
6. queue - Посмотреть очередь песен.
7. loop - Включить повторение песен.
